PFun Glucose Web API — Demo

Please note that the API is currently under active development

Endpoints:

  • View the interactive glucose modeling demo at /demo/run-at-time.
  • View the model parameters schema using the /params/schema endpoint.
  • Get the default model parameters using the /params/default endpoint.
